# Nuntius

## Descrição
Um aplicativo de chat em tempo real feito em node js.

## Principais Features e objetivos (WIP)
* Sistema de login
* Poder enviar mensagens de texto, imagens
* Chat entre dois clientes ou em grupos de clientes

## Tecnologias principais
* Node js
* React js
* Supabase
* Docker
* Webhooks
